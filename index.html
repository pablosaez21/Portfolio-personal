TFG: Meta-featuring para Meta-aprendizaje

Contexto

El meta-aprendizaje busca que los sistemas de IA aprendan cómo aprender mejor, usando el conocimiento acumulado de problemas anteriores para tomar mejores decisiones en nuevos problemas. El TFG se centra en la pieza clave: los meta-features.

Enfoque

Extracción y selección automática de meta-features que describan las propiedades estadísticas y estructurales de un dataset, con el objetivo de mejorar la recomendación de algoritmos en sistemas AutoML.

Estado actual

En desarrollo — Fase de experimentación y validación de resultados.

Tecnologías

Python scikit-learn Pandas Meta-learning AutoML
